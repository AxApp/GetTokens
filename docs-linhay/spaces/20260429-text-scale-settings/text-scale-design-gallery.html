How to Review
one variable only
1. 先横向看三栏，同一行内容是否仅因为字号变化而重新分层。
2. 重点看 `segmented control`、按钮、说明文字和卡片标题是否先拥挤。
3. 再切换主题，只判断气质，不要把主题差异误判成字号差异。
