Visual Thesis

Keep the Swiss Shell. Make Reading Feel Intentionally Larger.

这版稿子保留 GetTokens 现有的黑白红 mono 基因，但把设置面板做成偏 editorial 的控制页。用户一边切换文字大小，一边看到相同内容在默认和放大后的真实排版差异，避免“设置改了但不知道影响哪里”。

预览方式
建议默认展示“较大”档位的实时效果，并保留默认档位作为对照。这样用户不是盲调，而是直接理解“我会多出多少可读性，代价是多占多少空间”。
Implementation Notes
token first
1. 设置项只负责选档，不直接讲技术。
2. 预览页强调标题、正文、标签、按钮的联动变化。
3. “更大”档位建议只在预览中先验风险，再决定是否进首版实现。
