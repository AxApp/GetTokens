Option C / Warm Paper Console

Make Text Feel Kinder Without Losing the Operator Mood.

这版比前两版更柔和，适合你如果希望“可读性提升”不只是工程开关，而是顺手改善设置页的整体气质。它仍然保持 mono、边框、黑白红语义，但背景和层次更接近纸面工作台，减弱冷硬感。
